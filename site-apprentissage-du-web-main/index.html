Cours suivi

Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le Web est un service d'Internet qui permet d'accéder a des pages et ressources reliees entre elles via des liens.

Internet

Internet est un reseau mondial d'ordinateurs connectes qui echangent des donnees avec des protocoles communs.

Cloud

Le cloud designe des services informatiques (stockage, calcul, applications) heberges a distance et accessibles en ligne.

W3C

Le W3C est l'organisme qui definit les standards du Web pour garantir la compatibilite et l'accessibilite des sites.

Partie 2 - Distinguez les langages du Web

Langages client

Les langages client s'executent dans le navigateur et gerent l'affichage et l'interaction utilisateur (HTML, CSS, JavaScript).

Navigateurs

Un navigateur est un logiciel qui interprete les technologies Web pour afficher les pages et permettre la navigation en ligne.

Langages serveur

Les langages serveur executent la logique metier sur un serveur, traitent les requetes et preparent les reponses pour le client.

Frameworks

Un framework est une base de developpement avec des regles et outils qui accelerent la creation d'applications.

Bases de données

Une base de donnees stocke les informations de facon structuree pour les retrouver, les modifier et les securiser.

Site responsive

Un site responsive adapte automatiquement sa mise en page aux differents ecrans: mobile, tablette et ordinateur.

Application mobile

Une application mobile est un logiciel concu pour smartphone ou tablette, installe depuis un store ou le Web.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est une machine ou un service qui fournit des ressources, donnees ou applications a d'autres ordinateurs.

Adresse IP

Une adresse IP est un identifiant numerique unique permettant d'identifier un appareil sur un reseau.

Nom d’hôte et DNS

Le nom d'hote est un nom lisible (ex: site.com) et le DNS le traduit en adresse IP pour trouver le bon serveur.

Protocoles

Les protocoles sont des regles de communication entre machines, comme HTTP, HTTPS, FTP ou TCP/IP.

Partie 4 - Identifiez les métiers du web

Développeurs

Les developpeurs conçoivent, codent, testent et maintiennent les applications et sites web.

Product Managers et chefs de projets

Ils definissent les priorites, coordonnent les equipes et s'assurent du respect des objectifs, delais et besoins utilisateurs.

Designers

Les designers imaginent l'interface et l'experience utilisateur pour rendre les produits clairs, utiles et attractifs.

Testeurs

Les testeurs verifient la qualite du produit, detectent les bugs et valident le bon fonctionnement avant mise en production.

Techniciens informatique

Les techniciens informatiques installent, configurent et depannent les equipements, reseaux et logiciels.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

se rapportent à tous les éléments visuels d'un site web que vous pouvez voir

et avec lesquels vous pouvez interagir directement dans votre navigateur.

Le CSS sert a mettre en forme le HTML: couleurs, typographie, espacements, animations et adaptation aux ecrans.

JavaScript

Front-End

Langage de programmation utilisé pour rendre les pages web interactives et dynamiques.

JavaScript permet de reagir aux actions de l'utilisateur, manipuler le contenu de la page et communiquer avec des API.

JavaScript

Back-End

Langage de programmation utilisé côté serveur pour gérer la logique, les bases de données et les interactions avec le client.

Cote back-end, JavaScript est souvent utilise avec Node.js pour creer des API et des applications temps reel.

PHP

Back-End

Langage de programmation utilisé côté serveur pour développer des applications web dynamiques et interagir avec les bases de données.

PHP est largement utilise avec des CMS comme WordPress et se connecte facilement aux bases de donnees MySQL.

Java

Back-End

Langage de programmation utilisé pour développer des applications web robustes et évolutives.

Java est apprecie pour sa stabilite, sa securite et son ecosysteme, notamment avec le framework Spring.

Python

Back-End

Langage de programmation polyvalent utilisé pour le développement web, l'analyse de données, l'intelligence artificielle et bien plus encore.

Python est reconnu pour sa syntaxe claire et sa productivite, avec des frameworks web comme Django et Flask.

C#

Back-End

Langage de programmation utilisé pour développer des applications web, des jeux vidéo et des logiciels d'entreprise.

C# est souvent utilise avec .NET pour creer des applications web performantes et maintenables.

SQL

Bases de données

Langage de programmation utilisé pour gérer et manipuler les bases de données relationnelles.

SQL permet de creer des tables, inserer des donnees et executer des requetes pour analyser ou mettre a jour les informations.

Boîte à outils du développeur

  • IDE comme VS Code : Environnement de développement intégré permettant d'écrire, tester et déboguer du code.
  • Github : Plateforme de gestion de versions et de collaboration pour les projets de développement.
  • Librairies / bibliothèques : Collections de fonctions et de modules réutilisables pour faciliter le développement.
  • Frameworks : Structures de développement fournissant des outils et des conventions pour créer des applications rapidement. Exemples : React (JavaScript), Django (Python), Spring (Java).
